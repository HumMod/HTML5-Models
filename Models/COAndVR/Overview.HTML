Overview

On the average, cardiac output and venous return are equal. Each is determined by a unique set of factors.

Cardiac output (CO) is determined by the right atrial pressure (RAP), heart strength (HS), and shape of the Frank- Starling curve.

(1) CO = f (RAP) * HS

Venous return (VR) is determined by right atrial pressure (RAP), mean circulatory filling pressure (MCFP) and the resistance to venous return (RVR).

(2) VR = (MCFP - RAP) / RVR

We assume that cardiac output (CO) is equal to venous return (VR).

(3) CO = VR

So here is the challenge.

  • Use a value of RAP to calculate CO in Equation (1).
  • Use the some value of RAP to calculate VR in Equation (2).
  • Have the values of CO and VR satisfy the equality in Equation (3).
