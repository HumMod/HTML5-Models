These are the basic steps used to calculate the solution of any of the models in the Modeling Workshop.

The Model's Solution. Most models in this workshop have time as the independent variable. Solutions are calculated and displayed as a function of time. You obtain the desired solution by advancing the independent variable time.

Controlling The Solution. You control the solution using the buttons near the top of any Solve panel. There is typically one panel named Solve - Summary, but there can be more.

To advance the solution select an appropriate advance time from the drop down menu.

Then click the Go button. The solution will advance the specified amount of time and will automatically stop when done.

Click the Restart button to restart the solution. All variables (including time) will be reset to their intial values. The graphs will be cleared.

Click the Overlay button to get a restart without clearing the tracings on the line graphs. Using this feature, you can create multiple, overlayed solutions.
